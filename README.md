## Tech Stack

- **Framework**: Flutter (Null-Safe + State Management with Provider)
- **Backend**: Firebase / Node.js (configurable) / RestAPI
- **Real-time**: Firebase Firestore + Firebase Messaging
- **Live Streaming & Calling**: Agora.io
- **Maps**: Google Maps Flutter
- **Payments**: Stripe + PayPal
- **Local Storage**: GetStorage (high performance)
- **State Management**: Provider + ChangeNotifier
- **Animation**: Flutter built-in + Rive (soon)
